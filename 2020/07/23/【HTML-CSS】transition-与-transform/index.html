transition 过渡

  • transition-property: 规定设置过渡效果的CSS属性的名称

    使用：transition-property: width/height/color/all 指定宽/高/颜色/所有属性过渡

  • transition-duration: 规定完成过渡效果要多少秒或毫秒

  • transition-delay: 定义过渡效果何时开始。（既可以延迟（数值为正数）也可以提前（数值为负数））

  • transition-timing-function: 规定速度效果的速度曲线

    • Linear 线性变化
    • ease (默认值)
    • ease-in
    • ease-out
    • ease-in-out
    • cubic-bezier (http://cubic-bezier.com)

  • transition 复合样式： transition: all 1s 2s linear

    • duration时间在前，delay时间在后，只写一个默认duration
    • property不写默认all
    • timing-function不写默认ease

transform变形

  • translate 位移

    复合样式使用：transform: translate(100px, 0)

    单一样式：teanslateX, translateY, translateZ(3d)

  • scale 缩放：值是一个比例，正常大小为1，放大为原来的2倍是2，缩小为原来的一半是0.5，以此类的。会以当前元素中心点进行缩放。

    复合样式使用：transform: scale(2,0.5); (宽高缩放比例相同的时候可以只写一个值)

    单一样式：scaleX, scaleY, scaleZ(3d）

  • rotate 旋转：旋转的值单位是角度(deg)，弧度。正值为顺时针，负值为逆时针。

    复合样式使用：transform: rotate(40deg)

    单一样式：rotateX(3d), rotateY(3d), rotateZ

  • skew 斜切

    复合样式使用：transform: skew(40deg, 40deg)

    单一样式：skewX, skewY

  • transform的注意事项：

    1. 变形操作不会影响到其它元素。
    2. 变形操作只能添加给块元素，不能添加给内联元素。、
    3. 复合写法，可以同时添加多个变形操作，但执行是有顺序的，先执行后面的操作再执行前面的操作。
    4. 可以利用transform-origin设置基点。transform-origin：0 0

transform3D相关属性

  • rotateX(): 正值向上翻转

  • rotateY(): 正值向右翻转

  • translateZ(): 正值向前，负值向后

  • scaleZ(): 立体元素的厚度

3D写法：

  • scale3d(): 3个值 x y z 分别代表宽、高、厚度

  • translate3d(): 3个值 x y z

  • rotate3d(): 四个值 0/1(x轴是否添加旋转角度) 0/1(y轴是否添加旋转角度) 0/1(z轴是否添加旋转角度) deg

特殊属性

  • perspective: 离屏幕多远的距离去观察元素，值越大幅度越小。加在外部容器上。值越大，立体感越弱，值越小，立体感越强。

  • prespective-origin: 景深-基点的位置，观察元素的角度。

  • transform-origin: x y z Z轴只能写数值，不能写单词（transform-origin: center center -50px;）

  • transform-style: 3D空间

    • flat(默认值2D), prespective-3D(3D,产生一个三维空间)

  • backface-visibility: 背面隐藏

    • hidden, visible(默认值)
