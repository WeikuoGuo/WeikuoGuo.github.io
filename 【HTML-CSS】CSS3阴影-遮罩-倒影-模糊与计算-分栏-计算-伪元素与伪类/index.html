伪元素与伪类

CSS2.1中队伪类和伪元素的区别比较模糊。CSS3中对这两个概念做了较为清晰的解释，并且在语法上也做了明显的区分。

CSS3中规定伪类由一个冒号“：”开始，然后为伪类的名称；伪元素由两个冒号开始，然后为伪元素的名称。

伪类本质上是为了弥补常规CSS选择器的不足，以便获得更多信息。通常表示获取不存在于DOM树中的信息，或获取不能被常规CSS选择器获取的信息。

  • :hover/:focus/:empty ……

伪元素本质上是创建了一个有内容的虚拟容器，这个容器不包含任何DOM元素，但是可以包含内容。另外开发者还可以为伪元素定制样式。

  • ::selection(选中元素)/::first-line/::first-letter/::before/::after/……
