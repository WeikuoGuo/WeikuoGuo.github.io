【MySQL】CentOS7 SQL语句之 DDL，DML，DQL
DDL 数据定义语言库定义库名，库属性 创建库：CREATE DATABASE 库名 CHARSET utf8mb4; 库名小写，与业务有关，不要数字开头，库名不要太长, 不要使用保留字段。 必须定制字符集。 查询库: show databases; show create database 库名; (查询建库语句) 修改库：alter database 库名 charset utf8; 注意，修改后的字符集应当比修改前更大，是前者的严格超集，否则出现乱码。 删除库：drop database 库名; 注意：生产数据库中除了管理员没有任何人有删库权限。 ...
