响应式布局

利用媒体查询，既media queries, 可以针对不同的媒体类型定义不同的样式，从而实现响应式布局。

媒体类型：

取值 含义
all 用于所有设备
print 用于打印机和打印预览
screen 用于电脑屏幕，平板电脑，智能手机等
speech 应用于屏幕阅读器等发声设备

常见选项：媒体类型，and, not(取反), min-width, max-width, orientation:portrait(纵屏幕), orientation:landscape(横屏幕), <link>

示例：

多个条件同时满足的时候，后写的属性有较高优先级。

常见修改样式：

display, float, width。

注意：响应式代码写到要适配的CSS后面

Bootstrap

Bootstrap是目前最受欢迎的HTML，CSS和JS框架，用于开发响应式布局，移动设备优先的web项目。

特色：

  1. 响应式布局

  2. 基于flex的栅格系统

  3. 丰富的组件和工具方法

  4. 常见交互使用

官网：

https://getbootstrap.com/

Containers

containers: 版心布局

containers-fluid：通栏布局

Responsive breakpoints: 不同尺寸的响应节点

Grid system

  • Grid options: 网格配置, 默认将全部容器分为12个栅格，在不指定占有多少的时候默认自适应。

  • Responsive classes：响应式类

  • Gutters：间距，通过设置no-gutters类去掉间距。

  • Alignment：对齐方式

  • Recordering：排序

  • Offsetting: 间距

横向分辨率较大时横向排列，横向分辨率较小时纵向排列

插入换行标识

设置不同的屏幕分辨率下每行包含的列数不同。

Content

Reboot, Typography, Code, Images, Tables, Figures

参考文档：

https://getbootstrap.com/docs/4.5/content/

components

一些bootstrap中的常用组件，包括按钮，加载，徽章，幻灯展示等。

参考文档：

https://getbootstrap.com/docs/4.5/components/

Utilities

Borders, Clearfix, Close icon, Colors, Display, Embed

边框，清浮动，颜色，显示类型等设置

参考文档：
https://getbootstrap.com/docs/4.5/utilities/
